工具说明：

这是一个用来对学生进行随机抽题提问并打分的开源工具，可以通过导入现有的Excel题库来实现随机抽题

在使用Excel时，需要确保A列中只有题目，否则可能会导致软件崩溃

工具使用：
build.bat是用来将整个项目进行打包并转化为Windows可执行程序所使用的脚本工具
gui.py是用来对图形化界面功能定义
main.py是工具的主函数
config.py是工具的一些配置
data_manager.py是用来导入数据使用的

生成windows中的可执行文件后，可以使用导入题库的功能将Excel文件导入到工具中，导入后点击加载题库即可将题库导入到工具中
题库导入后可以设置抽题的数量，输入学生姓名后，点击随机抽题就可以开始随机抽题

每个题目可以通过滑动滑块来实现分数的评定，所有题目评定后保存考核结果会在目录中生成一个名为“考核结果.xlsx”的Excel的文件，里面会记录所有的问题和成绩
